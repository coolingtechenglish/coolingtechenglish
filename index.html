Essential Tech English Words

Key vocabulary organized by category to accelerate your learning

🛠️ Development

Repository A storage location for code
Deploy Release code to production
Refactor Restructure existing code
Debug Find and fix errors
Iterate Repeat a process to improve
Compile Convert code to machine language

☁️ Infrastructure

Scalability Ability to handle growth
Latency Delay in data transfer
Throughput Amount of data processed
Redundancy Backup for fault tolerance
Provision Set up resources for use
Orchestrate Coordinate multiple services

👥 Collaboration

Stakeholder Person with project interest
Blocker Issue preventing progress
Milestone Significant project checkpoint
Scope Extent of project work
